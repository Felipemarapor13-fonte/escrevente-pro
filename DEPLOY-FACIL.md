# 🚀 Deploy Súper Fácil - Netlify Drop (Sem Git!)

## Opção 1: Netlify Drop (MAIS FÁCIL - 30 segundos)

### Passo a Passo:

1. **Acesse:** https://app.netlify.com/drop

2. **Arraste a pasta inteira:**
   - Pegue a pasta `GUIA_ESCREVENTE_PRO`
   - Arraste para a área indicada no site
   - Aguarde o upload (10-30 segundos)

3. **Pronto!**
   - Seu site estará no ar imediatamente
   - URL: `https://random-name-12345.netlify.app`

4. **Personalize o nome:**
   - Clique em "Site settings"
   - "Change site name"
   - Digite: `aprova-sp` (ou outro disponível)
   - URL final: `https://aprova-sp.netlify.app`

### Vantagens:
- ✅ Não precisa de GitHub
- ✅ Não precisa instalar Git
- ✅ Não precisa de comandos
- ✅ Publicado em 30 segundos
- ✅ HTTPS automático
- ✅ Responsivo funciona perfeito

### Atualizar o site:
1. Volte em https://app.netlify.com/drop
2. Arraste a pasta atualizada
3. Substitui automaticamente!

---

## Opção 2: Vercel (Também muito fácil)

1. **Acesse:** https://vercel.com/new

2. **Importe do GitHub:**
   - Faça login com GitHub
   - Selecione o repositório `guia-escrevente`
   - Clique em "Deploy"

3. **Pronto!**
   - URL: `https://guia-escrevente.vercel.app`

---

## Opção 3: GitHub Pages (Com Git)

Siga o arquivo `DEPLOY.md` que está na pasta do projeto.
